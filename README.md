# Clause

Clause is a grammar escape-room platform for Grades 6–9. The current MVP is an interactive, deterministic demo of the teacher workflow, student mission, feedback, appeal, and dashboard experience.

## Run locally

```bash
npm install
npm run dev
```

Open [http://localhost:3000](http://localhost:3000).

## Verify

```bash
npm test
npm run lint
npm run build
```

The guest-room regression test covers the sample journey from landing page through final lock completion, including retry feedback and an appeal.
